haxe-tjsonStyleCl
=========================

Styling for library [TJSON](https://github.com/martamius/TJSON) with indents like a:<br/>

```
{
	"str": "value",
	"num": 1231231,
	"arr": [
		"item 1",
		"item 2",
		"item 3"
	],
	"obj": {
		...
		...
		...
	},
	...
	...
	...
}
```

## Install:
`haxelib git tjsonStyleCl https://github.com/r3d9u11/haxe-tjsonStyleCl.git`
